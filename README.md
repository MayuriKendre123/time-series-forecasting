## Project Overview
This project focuses on forecasting future values using time series data.
ARIMA and LSTM models are implemented and compared.

## Dataset
The dataset contains daily sales data with Date and Sales attributes.

## Technologies Used
- Python
- Pandas, NumPy
- ARIMA
- LSTM
- TensorFlow
- Jupyter Notebook

## Results
LSTM model provides better accuracy compared to ARIMA.

## How to Run
1. Upload files to Google Colab or Jupyter Notebook
2. Install requirements using `pip install -r requirements.txt`
3. Run the notebook
